# Цифровая обработка изображений
## Разработка Windows-приложения
### Реализованная функциональность:
- GUI 
- выбор цветового обрабатываемого изображения
- возможность настройки параметров обработки
- отображение результата обработки с возможностью его сохранения
- отображение на дисплее координат и яркости любого выбранного пикселя и яркости соответствующего ему пикселя обработанного изображения
- гистограммы яркости изображений и разрезы функций яркости изображений по строкам и столбцам

### Примечания
#### Обработка границ
Для решения проблемы обработки границ изображения был использован метод зеркального дополнения изображения (reflect padding).
``` python
pad_before = window_size // 2
pad_after = window_size - pad_before - 1

padded = np.pad(
    img,
    ((pad_before, pad_after), (pad_before, pad_after)),
    mode="reflect"
)
``` 

#### Возможное деление на нулевую яркость
Для решения проблемы деления на нулевую яркость при вычислении коэффициента масштабирования используется небольшая константа 1e-5.
``` python
ratio = y_processed / (y + 1e-5)
```

#### Выход значения яркости из заданного диапазона
Для предотвращения выхода значений яркости за допустимый диапазон [0,255] применяется операция ограничения (clipping).
После вычисления коэффициента масштабирования ratio новые значения цветовых каналов корректируются и ограничиваются с помощью функции np.clip, что гарантирует сохранение значений в допустимых пределах.
``` python
r_new = np.clip(r * ratio, 0, 255)
g_new = np.clip(g * ratio, 0, 255)
b_new = np.clip(b * ratio, 0, 255)
```
